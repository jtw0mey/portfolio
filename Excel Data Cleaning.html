    Data Cleaning

    Cleaning data in an Excel file



    P r o j e c t
    A dataset about US Presidents has numerous mistakes and errors. If the data has inconsistencies and errors, it is likely the results will be flawed when you go on to do further analysis. I will be cleaning this file and making the data standardized and in usable formats.

    D a t a    C l e a n i n g
    I first checked if there were any duplicate rows using Excel's remove duplicates function, it found 2 duplicate rows for Barack Obama and Woodrow Wilson. These were removed.

    I then used the =PROPER function to capitalize the first letters of each President's names. This capitalizes the first letter in a text string and any other letters in text that follow any character other than a letter. I did this to standardize the names in a uniform format.
    E.g. 'franklin d. roosevelt'  was transformed to 'Franklin D. Roosevelt'.

    There is also multiple cells that have excessive spaces in them. I used the =TRIM function to remove these extra spaces.
    E.g. 'U.S. Senator        (   Class 1   )      from    Indiana      (1881-1887)'  was transformed to 'U.S. Senator (Class 1) from Indiana (1881-1887)'.

    I also used filters to see if there were any spelling mistakes or multiple words used for the same meaning.
    E.g. 'Republican'  and  'Republicans'  are both used in the Party column.
    I used the Find & Replace function to change them both to Republican to standardize the column's data.

    For the salary column, I changed the number format from currency to number to removed the $ sign. If the numbers remained with the $, it would read as a string/text if I were to do further analysis on other platforms like SQL etc.
    E.g. '$35000.00'  was transformed to '35000'.
