    P r o j e c t

    I will be using historical stock market data from S&P 500 between 2014 - 2017 to create a dashboard visualising the stock volume by data, day of week and company. I will also be displaying Amazon's stock volatility and finding out which day had the most volatility.



    D a s h b o a r d

    I built this dashboard using Tableau. I used line charts and a bar graph to clearly display the trade volume by stock and day of week. I used a line chart for the volume by date, as it is great for displaying changes over time, and I was easily able to show which date had the highest trade volume.
    For Amazon's volatility, I was able to create a candlestick graph, which displays the relationship between Amazon's high, low, opening, and closing prices. This was great for visualising volatility with the biggest candle bar showing which day was the most volatile.



    What I Learnt:

    • How to create a candlestick graph on Tableau using calculated fields to display volatility.

    • How to use a line graph using a double axis to display the high and low price.
